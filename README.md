# Agentic AutoML Platform

An autonomous machine learning platform built using LangGraph, LangChain, Scikit-learn, XGBoost, Optuna, and Streamlit.

## Features

* Dataset Profiling Agent
* Data Quality Agent
* Data Preparation Agent
* Model Selection Agent
* Training Agent
* Evaluation Agent
* Hyperparameter Tuning Agent
* Feature Engineering Agent
* Experiment Tracking Agent
* Decision Agent
* LLM Critic Agent
* Explainability Agent

## Workflow

Dataset → Profiling → Data Quality → Data Preparation → Model Selection → Training → Evaluation → Tuning → Feature Engineering → Decision → Reporting

## Tech Stack

* Python
* LangGraph
* LangChain
* Scikit-learn
* XGBoost
* Optuna
* Streamlit
* Ollama (Qwen 2.5)

## Results

### Titanic Dataset

* XGBoost: 82.6% Accuracy
* Random Forest: 82.4% Accuracy
* SVM: 82.0% Accuracy

### Iris Dataset

* SVM: 96.7% Accuracy
* Logistic Regression: 95.8% Accuracy

## Run

```bash
pip install -r requirements.txt
streamlit run frontend/streamlit_app.py
```
